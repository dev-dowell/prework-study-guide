HTML

  • The Head element contains information about the webpage
  • The Body element represents the visibile content, seen by the user
  • Headers are defined by h1 to h6, h1 being the biggest
  • Only one h1 per page
  • ul = unordered list, ol = ordered list, li = the items on the list

CSS

  • A margin indicates how much space around an element
  • Padding indicates how much space around the content inside an element
  • colours can either be designated by name or by using their HTML Hecidecial code
  • Cobine lines of code to create DRY (do not repeat yourself) code files

Git

  • git status: checks which branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: checks we are working the most recent version of the main
  • git add -A: commits all files in the working branch to the staging area
  • git commit -m: commits the files in the staging area, -m allows us to add a message
